# Formatted CV Skill

将候选人简历整理为 IntelliPro Formatted CV。支持 Windows 和 macOS，默认沿用原简历语言。

## 安装并使用

把候选人简历上传到 Codex，然后发送：

> 请安装此链接中的 Formatted CV Skill，安装完成后使用它处理我上传的候选人简历：https://github.com/rawpaper123/formatted-cv-skill/tree/main/formatted-cv

安装后，在同一个 Codex 任务中再次上传简历，直接说“请使用 Formatted CV Skill 处理这份简历”即可。

仓库中的 DOCX/PDF 参考模板已经脱敏，仅保留版式、品牌元素和结构占位说明。
